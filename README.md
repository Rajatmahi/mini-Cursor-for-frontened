# Mini Cursor AI

A terminal-based AI coding assistant inspired by Cursor AI.

This project can:

- Create folders automatically
- Create HTML/CSS/JS files
- Generate frontend websites
- Execute terminal commands
- Build mini projects using AI

---

# Features

- OpenRouter API
- AI Website Generator
- Automatic File Creation
- Terminal Command Execution
- Node.js Based
- Beginner Friendly

---

# Tech Stack

- Node.js
- OpenAI SDK
- OpenRouter
- readline-sync
